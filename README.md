# Paleta de Colores

Una página web interactiva que genera paletas de colores aleatorias y permite cambiar el formato de los valores entre `HEX` y `HSL`.

## Descripción

Este proyecto muestra una paleta de colores dinámica donde el usuario puede:
- generar una nueva paleta
- seleccionar entre 6, 8, 9 o 25 colores
- ver los valores en formato `HEX` o `HSL`

## Características

- Generación aleatoria de colores
- Cambio de formato de color en tiempo real
- Interfaz minimalista y responsiva
- Contenedor de tarjetas dinámico que se actualiza al cambiar la paleta

## Estructura del proyecto

- `index.html` - estructura de la página
- `css/styles.css` - estilos principales
- `css/normalizador.css` - reset / normalizador
- `js/script.js` - lógica de generación y actualización de colores
- `img/` - recursos de imágenes utilizadas en la interfaz

## Cómo usar

1. Abrir `index.html` en el navegador
2. Seleccionar la cantidad de colores deseada
3. Presionar `Generar Paleta`
4. Cambiar entre `HEX` y `HSL` para ver los valores de color
